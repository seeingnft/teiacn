# Teia 是

……由艺术作品构成的协作艺术，一个分享想法与创意作品的空间。

……一个非营利、开源的在线平台，展示基于 Tezos 区块链的数字资产 NFT。

……一个集体，秉承可持续性、可及性与公平的价値理念。

……由社区拥有和维护，由参与者共同建设和完善。

……一个永无止境的进行时作品。
